## 教程
此教程是为了不知道如何进入我们的MTR服务器的人所准备的。

请准备：
- 一台电脑
- 一个启动器

### 1.下载游戏及模组
打开启动器，下载一个新版本，版本号:`1.20.1`、模组加载器：`Forge`，下完后请下载[必要模组](http://leonmmcoset.jjmm.ink:1000/web/mod/mod.zip)并解压到版本的`mods`文件夹。

#### 资源包
首先，要下载必备资源包：[链接](http://leonmmcoset.jjmm.ink:1000/web/mod/gbt.zip)

要下载额外的上海资源包（低配慎装）：[链接](http://leonmmcoset.jjmm.ink:1000/web/mod/shanghai.zip)

你也可以下载其他资源包：[链接](http://leonmmcoset.jjmm.ink:1000/web/mod/more.zip)

::: tip
必备资源包和上海资源包不用解压，直接装进`resourcepacks`文件夹

其他资源包把zip文件解压（不用解压zip里面的zip文件），解压到`resourcepacks`文件夹
:::
::: warning
要关注群消息，及时更新资源包！
:::

### 2.进入游戏
启动后，打开`多人游戏`、`添加服务器`，IP：`leonmmcoset.jjmm.ink:7180`，点击`完成`，进入服务器。

等待Mojang标加载完成，那么 **恭喜你！** 你成功的进入了服务器！
