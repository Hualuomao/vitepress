# Leon轨道交通线路标准
## 前言
由于我服是一个自由服务器，所以不会有太多标准，但是为了我服不会像~~一坨屎~~一样，所以制定了一个标准，如不符合这个标准会被判为低创。

## 站点
要有出入口，要有线路，要有中英文名（格式：`中文|英文`），要有屏蔽门（火车和高铁和公交除外），要有闸门。

## 线路
要有中英文名和上下行（格式：`中文|英文||上下行`），要有墙壁（不要穿膜，典型错误线路：信安1号线），要有桥梁，要有相对应的类型（高铁/火车：高铁，地铁：地铁，公交/轻轨：轻轨），不要堵车。
