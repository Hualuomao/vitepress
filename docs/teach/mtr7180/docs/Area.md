# 区域规定
## 规定
### 顺序（从上往下）
#### 国家管控
- 市（或县）
- 区
- 社区

#### 自治区
- 自治区
- 市
- 区
- 社区

### 自治区规定
自治区由公司私信服主申请，申请成功后请在边界造围墙或界碑等代表自治区边界的建筑。

自治区只允许申请自治区的公司和申请自治区的公司授权的公司建造。

自治区可以在Leon轨道交通法则和标准之下指定自己的规则和标准，但是如果自定的规则和标准对Leon轨道交通法则和标准有冲突，皆按照Leon轨道交通法则和标准中的内容处理。

自治区管理权的规则的最终解释权归申请自治区的公司所有。
