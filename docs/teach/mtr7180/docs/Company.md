# 公司规定
## 什么是公司
公司是一个可以控制并改造线路或车站或建筑的一个群体，

每个公司不可互相干扰。
## 规定
- 任何公司都是`Leon轨道交通城市建设与轨道交通建设公司`的分公司，任何公司都要听总公司的命令，否则关门大吉。
- 任何公司都有权保护自己的建筑，任何公司都不可干扰其他公司进行建造，任何公司都不可更改他人的建造（除建筑所有者公司授权的以外）
- 任何公司可以与其他公司合作，但是不可私下合作，任何合作都要公开
- 公司可以开会，但是要和总公司说明开会内容或邀请总公司的一位代表（比如我，LeonMMcoset）聆听会议

---

此规定所有解释权归Leon轨道交通管理组所有
