# Leon服务器开服网
::: warning
此开服网是给那些没钱买服务器的人准备的，如果你有服务器，请不要使用！
:::
Leon开服网是由`LeonMMcoset`使用[MCSM（MCSManager Panel）](https://docs.mcsmanager.com/zh_cn/)开发的开服网。

## 使用规定
1.不能修改实例设置中的启动命令（修改完了重启可能打不开）。

2.严禁上传非上报的`*.exe`、`*.com`、`*.cmd`、`*.bat`等可执行文件或Batch脚本文件。

3.不可更改密码，管理员给你的密码保存好，不许泄露用户名、密码。
